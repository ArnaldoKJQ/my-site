## Why I built my own blog

Most blog platforms feel like overkill for what I need. I wanted something I fully own — no CMS subscription, no vendor lock-in, just plain files in a GitHub repo.

So I built a small Node.js static site generator. Posts are Markdown files, `build.js` turns them into HTML, and GitHub Actions deploys on every push.

## The stack

- **Node.js** — `build.js` reads `.md` files and generates the full site
- **GitHub Pages** — free hosting, custom domain support
- **Cloudflare Workers** — handles the admin API, LinkedIn OAuth, and AI summaries
- **`marked`** — Markdown → HTML
- **`gray-matter`** — YAML frontmatter parsing
